My current research focuses on the optimization of multi-modal transport. The idea of multi-modal routing is to adapt the famous shortest path problem to road networks combining several modes of transport. One way of achieving this is to construct the network as a layered structure, where each layer corresponds to a mode of transport. Thus, the ultimate goal of this research will be to find a solution to this mobility problem by including the notions of time, price and ecological impact. The second underlying objective of this research is to enable users of car-sharing solutions to optimize their trips. This is where this project comes into its own for the common good, by designing algorithmic solutions to ultimately reduce the ecological impact of transport, first in Brussels and then, we hope, everywhere.

From Movement to Melody

This project merges human movement with AI through computer vision to create a dynamic dance experience with responsive sound generation, enriching both performer and audience experiences and contributing significantly to cultural innovation.

Random Music Sequencer

In this Arduino project, I'm crafting a compact 16-bar sequencer. I'm programming steps and notes to play in a sequence, controlling the timing and order of musical events. Adding a twist, I'm incorporating random parameters to introduce variation and spontaneity into the composition.

Rap Lyrics Creation

This project utilizes embeddings to create a visual creationn for French rap lyrics, where smaller distances between embeddings reflects closer proximity, providing a compact visualization of thematic connections and stylistic similarities among the lyrics.

Hands-Free Blog Generation

A framework for automating the creation of blogs and articles using LLMs, focusing on time-saving and customization for diverse topics. It outlines the architecture, methodology, and practical applications of the system, highlighting its potential to streamline content generation while maintaining quality and personalization.

This project provides a Python template for building esoteric programming languages. It includes a lexer and an interpreter, and a main file that ties everything together.
